# LayerZip Repository.

LayerZip is a program independent way to store 2d image layers using zip deflate, png, and a json file.

[this is just the repository](https://github.com/DNSCond/layerzip), the real spec is at
[https://antrequest.nl/layerzip/, this link will take you to the latest version](https://antrequest.nl/layerzip/)
